# 净值 — 个人资产负债表

一个纯本地的个人资产负债表工具，支持多币种、多账户，实时追踪净资产变化。

**在线使用：** [https://balancesheet-cnt.pages.dev](https://balancesheet-cnt.pages.dev)

## 主要特性

- 资产/负债分类管理
- 多币种支持（CNY/USD/HKD/SGD），自动汇率换算
- 净资产趋势图，每小时快照
- 转账、还款、外汇等多种操作类型
- 隐私模式，隐藏金额但保留图表
- 数据纯本地存储（IndexedDB），无需注册
- PWA 支持，添加到主屏幕如原生 App
- 导出/导入 JSON 备份

## 技术栈

Next.js + TypeScript + Tailwind CSS + Dexie.js + Recharts

## 本地开发

```bash
pnpm install
pnpm dev
```
